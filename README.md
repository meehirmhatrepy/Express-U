## Run Locally

Clone the project

```bash
  git clone https://github.com/palak2665/Express-U.git
```

Go to the project directory

```bash
  cd Express-U
```

Install dependencies

```bash
  pip install -r requirements.txt
```

Go to the model directory

```bash
  cd Model/third_model
```

Run

```bash
  runipy raw_model.ipynb
```
